# Khatha Wallet Frontend

## Overview

Khatha Wallet is a modern web application that helps retailers manage customer accounts, track transactions, and maintain digital ledger records. This frontend provides an intuitive user interface for managing customers, balances, and transaction history.

## Features

* Retailer Login Interface
* Customer Management Dashboard
* Add and View Customers
* Track Credits and Debits
* Transaction History Management
* Responsive User Interface
* Real-time Balance Updates

## Tech Stack

* React.js
* Vite
* JavaScript
* CSS
* Material UI (if used)

## Installation

### Clone the Repository

```bash
git clone https://github.com/VDineshKumar18/Khatha-Wallet.git
```

### Navigate to Frontend

```bash
cd frontend
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

## Usage

1. Start the frontend application.
2. Login as a retailer.
3. Add customers and manage accounts.
4. Record money given or received.
5. View balances and transaction history.
