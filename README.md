# Khatha Wallet

A digital wallet and customer account management system designed for retailers to track customer transactions, manage credit/debit records, and maintain digital khatha (ledger) books efficiently.

## 📌 Overview

Khatha Wallet helps shop owners maintain customer records digitally instead of using traditional notebooks.

The application allows retailers to:

- Add and manage customers
- Record money given and received
- Track transaction history
- View outstanding balances
- Maintain digital khatha records
- Access customer details anytime

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/VDineshKumar18/Khatha-Wallet.git
cd Khatha-Wallet
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

---

## 🔄 Workflow

1. Retailer logs into the system.
2. Adds customers.
3. Records money given or received.
4. System updates customer balance.
5. Transaction history is stored.
6. Retailer can view reports and balances anytime.
